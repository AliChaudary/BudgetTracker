

## âœ… Running the App

From the **root directory**, run:

```bash
npm run dev
```

This will:

- Start the **Express backend** on `http://localhost:3001`
- Start the **Vite frontend** on `http://localhost:5173`

Make sure `nodemon` is installed (it should be added in `devDependencies`).
